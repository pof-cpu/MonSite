Time series

On importe un dataframe d'un fichier csv.
Dans la méthode read_csv() l'indexation se fait sur la colonne "Date" grâce à l'option index_col='Date'.
Pour transformer les dates de type object à type date on utilise l'option parse_dates = True 

                                
                                    import pandas as pd
                                    data = pd.read_csv(
                                        'fichier.csv', 
                                        sep=';', 
                                        index_col = 'Date', 
                                        parse_dates = True
                                        )
